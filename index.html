Mymymymy Persona!

An AJAX authentication system built around Mozilla Persona.

Mymymymy Persona! is a client-server AJAX-wrapper that makes it easy incorporate Mozilla Persona into existing web applications. If you have any questions feel free to contact me via my blog or github account.

How it works


The mpc client object interacts with Mozilla Persona's navigator object and sends AJAX form posts to server.php. The $mps server object processes the client AJAX posts, interfaces with Mozilla Persona's verification service, and sends formatted JSON response packets back to the client.

System requirements


The Mymymymy Persona! client code should run in any modern web browser. At the time of this writing, it's been tested in Firefox 17.0 and Chromium Version 20.0.1132.47 for Ubuntu.

The Mymymymy Persona! server code was written and tested on a CentOS 6 Linux system running Apache 2.2 and PHP 5.3.3.

Client code


Include the Mymymymy Persona! javascript files in your site's <head> tag (omit the jquery-1.7.2.min.js include if your site already uses jQuery): 
<script type="text/javascript" src="js/jquery-1.7.2.min.js"></script>
<script src="https://login.persona.org/include.js"></script>
<script type="text/javascript" src="js/MymymymyPersona.class.js"></script>

Add the following code to jQuery's $(function(){// Put mpc code here.}); ready event to instantiate and run the MymymymyPersonaClient (mpc) object: 
mpc = new MymymymyPersonaClient();
mpc.run();

Place the following <div> tag in your HTML source where you'd like the Sign In / Sign Out buttons to appear: 
<div id="personaBar"></div>

Server code


Include the MymymymyPersona.class.php file: 
include_once('lib/MymymymyPersona.class.php');

Instantiate the MymymymyPersonaServer ($mps) object, and call the $mps->run() function: 
$mps  = new MymymymyPersonaServer($_SERVER['SERVER_NAME'], $_SERVER['SERVER_PORT']);
$mps->run();

At the end of your PHP script / application, you have the option of calling the $mps->close() function: 
$mps->close();
